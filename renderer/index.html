DAoC Qbind Editor

How to use

Layout: Each bar shows one bank (10 slots) at a time, like in-game. Click the numbered chips above the row to switch banks. A small green dot under a bank number means it has bindings. Click Expand all banks for the full 10×10 grid view.

Editing: Click any slot to bind. Press the key combo (with Shift, Ctrl, or Alt). Right-click a slot to clear it instantly.

Profiles: Save your current binds + macros as a named profile (e.g. "PvE Reaver", "BG Crafter"). Click any profile in the sidebar to apply it to the current character. Click Apply to characters… to push a profile to multiple characters at once.

Folder: The editor auto-finds your eden folder on first launch. Use File → Change eden folder… if you have a non-standard install or multiple freeshards.

Tip: The yellow banner at the top warns you if DAoC is running. Always close the game before saving — the client overwrites the INI on logout.
